# Контекст репозитория Notifications

Этот файл содержит общие правила для всех подпроектов уведомлений в репозитории.

## Чаты Пачки

В репозитории используются два типа чатов Пачки:

- Тестовый чат: `39363429`.
- Продовый чат: `36815841`.

Продовый чат `36815841` сейчас используется для действующих FBO-уведомлений.

Новый подпроект `/действия_уведомление` до согласования использует тестовый чат `39363429`.

Общее правило для новых подпроектов:

1. Сначала логика сбора данных и формат уведомления оттачиваются в диалоге без автоматической отправки.
2. Затем тестовые отправки делаются только в тестовый чат `39363429`.
3. После подтверждения формата и логики подпроект можно переключать на продовый чат `36815841`.

## Безопасность

- Не коммитить токены, JSON service account, env-файлы и сгенерированные отчеты.
- Chat id для каждого подпроекта должен храниться в отдельном secret или явно документироваться.
- Продовый чат нельзя использовать для тестовых прогонов без явной команды.
