# WB Marketing Notifications

Подпроект для ежемесячного уведомления по заполненности контента WB.

## Что делает отчет

- Собирает товары, где план продаж за текущий месяц > 10 или FBO-остаток > 10.
- Проверяет статусы карточки через открытый WB basket `card.json`: рекомендации продавца, рич-контент и видео.
- Формирует три Markdown-файла:
  - `Настройки Рекомендаций Продавца.md`
  - `Незаполненный контент по Артикулам.md`
  - `Полный контент по Артикулам.md`
- Отправляет основное сообщение в Пачку с тремя файлами.
- Создает тред к сообщению и отправляет туда два отдельных сообщения:
  - описание файлов;
  - сводку по маркетологам / кабинетам / ошибкам.

## Запуск

```bash
npm ci
npm run send
```

Обязательные env vars:

- `ABCAGE_ANALYZER_TOKEN`
- `PACHCA_TOKEN`
- `PACHCA_CHAT_ID`

## Автоматизация

GitHub Actions workflow: `.github/workflows/wb-marketing-notifications.yml`.

Cloudflare Worker: `cloudflare/wb_marketing_notifications`.

Расписание: каждое 20 число месяца в 13:00 МСК.

Ручная команда в Пачке: `/контент_уведомление`.
